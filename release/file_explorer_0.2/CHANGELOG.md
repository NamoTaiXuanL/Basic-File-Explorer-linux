# 变更日志 (Changelog)

作者：文件浏览器项目组Seraphiel

## v0.2.0 — 2025-11-17

### 🎨 界面和用户体验
- **多视图模式支持**：实现详细信息、大图标、小图标三种显示模式
- **图标完美对齐**：修复图标和文字的对齐问题，提供专业的视觉效果
- **UTF-8安全处理**：修复中文字符截断错误，确保文件名正确显示
- **菜单栏自动刷新**：点击"转到"功能时自动刷新内容，无需手动操作

### 🎯 智能文件图标系统
- **EXE文件图标**：集成可执行文件专用图标（25px/50px）
- **DLL文件图标**：集成动态链接库专用图标（25px/50px）
- **TXT文件图标**：集成文本文件专用图标（25px/50px）
- **代码文件图标**：集成60+种编程语言通用图标（25px/50px）
  - 支持语言：rs, py, js, ts, jsx, tsx, java, c, cpp, cc, cxx, h, hpp, hxx, cs, vb, php, rb, go, swift, kt, scala, clj, hs, ml, fs
  - 脚本语言：sh, bat, cmd, ps1, pl, lua, awk, sed, vim, fish, zsh, bash, ksh, tcsh, csh
  - Web技术：html, htm, css, scss, sass, less, json, xml
  - 配置文件：yaml, yml, toml, ini, cfg, conf, log
  - 文档格式：md, markdown, r
- **无格式文件图标**：为无后缀名文件提供专用图标（25px/50px）
- **默认文件图标**：为其他未定义文件类型提供通用图标（25px/50px）

### 🖼️ 应用程序品牌
- **自定义应用图标**：集成专业的应用程序图标
- **多尺寸支持**：图标在标题栏、任务栏、Alt+Tab界面完美显示
- **模块化管理**：图标功能完全模块化，便于维护和扩展

### 🔧 核心功能增强
- **复制粘贴修复**：修复菜单栏复制粘贴功能冲突问题
- **文件操作完善**：改进复制、粘贴、新建文件夹等操作逻辑
- **帮助系统**：添加完整的帮助系统和关于对话框
- **隐藏文件支持**：可选择是否显示系统隐藏文件

### 🏗️ 架构优化
- **模块化重构**：将图标相关代码分离到独立模块，提高可维护性
- **代码质量提升**：优化代码结构，减少耦合度
- **错误处理增强**：完善文件操作和图标加载的错误处理

## v0.1.0 — 2025-11-15

- 文件列表文字左对齐，整行点击区域与选中高亮
- 内容框新增列头与可拖拽分割线，列宽动态比例，列内文本裁剪
- 目录框文本按行裁剪，避免长名称溢出到内容区
- 目录与内容列表文件夹使用彩色图标（矢量绘制），提升可辨识度
- 内容标题替换为四大导航按钮（返回/前进/刷新/主页），新增导航历史支持
- 重写贯穿式三栏标题（目录/内容/预览），分割线齐平，整体对齐
- 工具栏"列表/图标/详情"按钮样式与高度统一，视觉一致
- 修复若干点击区域与对齐问题，提升稳定性
